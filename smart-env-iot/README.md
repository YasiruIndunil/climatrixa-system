# Smart Env IoT

ESP32 sensor firmware — planned component.

Status: not yet started. Will contain firmware for DHT22, MQ-135, and BME280 sensors publishing to the backend via MQTT, as described in Section 1.4.1 of the project report.

Until hardware is available, the backend can be developed and tested against the simulated data generator in `smart-env-backend/scripts/seed_data.py`.
